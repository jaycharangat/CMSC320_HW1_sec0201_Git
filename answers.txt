Q1: Why might we consider using Git?

Answer: Using Git is helpful when multiple people are working and modifying the same files at the same time. It makes it easy to push and pull changes and makes it easy to keep track of the history of the files which helps if you want to go back to an earlier version.

Q2: What field could data science be applied to? Give an Example.

Answer: Data science could be applied to the marketing field as it could help determine which advertisements will be more efficient by conducting experiments to see which ones people respond to more. It could also be applied to the financial field as it can help determine if a company is doing well or not and can be used to look at individual products to see where problems could be coming from.

Q3: What is the difference between Merging and Rebasing?

Answer: The difference between merging and rebasing is that merging saves the history of the repository while rebasing will make the history linear by rewriting it.
